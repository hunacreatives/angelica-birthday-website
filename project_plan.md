# Gel's 30th Birthday Invitation

## 1. Project Description
A soft, elegant digital birthday invitation website for Gel's 30th birthday celebration. The site features a feminine, warm, and intimate scrapbook-style design with pastel colors, watercolor florals, and handwritten typography. The goal is to create a cozy, sentimental, and celebratory digital experience that feels like a personal keepsake.

## 2. Page Structure
- `/` - Home (Single-page invitation with all sections)
  - Hero Section
  - Event Details Section
  - Message Section
  - RSVP Section
  - Personal Note Section

## 3. Core Features
- [ ] Hero with polaroid-style photos and decorative elements
- [ ] Event details display (Date, Time, Location)
- [ ] RSVP form with form submission
- [ ] Personal note / message section
- [ ] Gentle animations (fade-ins, floating elements)
- [ ] Mobile-first responsive design

## 4. Data Model Design
No database needed for this invitation site. RSVP data will be collected via form submission.

## 5. Backend / Third-party Integration Plan
- Supabase: Not needed
- Shopify: Not needed
- Stripe: Not needed
- Forms: RSVP form via get_form_url

## 6. Development Phase Plan

### Phase 1: Design & Home Page Build
- Goal: Build the complete single-page invitation with all 5 sections
- Deliverable: Fully styled and animated home page with Hero, Event Details, Message, RSVP, and Personal Note sections

### Phase 2: Polish & Interactions
- Goal: Add RSVP form integration, refine animations, ensure mobile responsiveness
- Deliverable: Production-ready invitation with working RSVP form and smooth interactions
